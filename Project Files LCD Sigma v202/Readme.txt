To Run BCN3D project is necessary to have extra button that you will find them at the forlder ExtraFiles.

Copy the button folders to *\4D Labs\4D Workshop 4 IDE\SysImages\Buttons, for instance C:\Program Files (x86)\4D Labs\4D Workshop 4 IDE\SysImages\Buttons.

After that you will see more 4Dbuttons at 4DWorkshop.
